## 必要そうな情報

- `firestore.rules` は使ってます
    - `npm run deploy:firestore` でデプロイできます
- `functions` は使ってます
    - functionsにcdした後で`npm run deploy` でデプロイできます

あとはまだ勉強中なのでテンプレのままです。
